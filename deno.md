# Why deno?

::left::

<Youtube id="M3BM9TB-8yA" />

Ryan Dahl - JSConf EU 2018

::right::

- Node async is bad (no promises)
- Security (access to all syscalls)
- Build System (gyp)
- package.json (boilerplate)
- node_modules (too complex)
- require without extension
- index.js (unnecessary)
